#💰 Monthly Expense Invoice Generator

A clean, responsive, and printable **Monthly Expense Invoice Generator** built using **HTML, CSS, and Vanilla JavaScript**.
This application helps users record daily expenses, calculate monthly totals automatically, convert the total amount into words, and generate a printable invoice.

## ✨ Features

- 📅 Select any month
- 👤 Add "Prepared By" name
- ➕ Add or remove expense entries dynamically
- 💵 Automatic expense calculation
- 🧾 Converts amount into words
- 💾 Auto-save using Local Storage
- 🖨️ Print / Download invoice
- 📱 Responsive Design
- 🎨 Elegant modern UI
- ⚡ Lightweight (No frameworks)

---

## 💡 Features Explained
### ✔ Dynamic Rows

Users can add and delete expense entries.

### ✔ Auto Calculation
Updates total instantly while typing.
### ✔ Amount in Words

Example:

```
₹1250
One Thousand Two Hundred Fifty Rupees Only
```

### ✔ Auto Save

The application automatically stores data in browser Local Storage.
Refreshing the page will not lose your work.

### ✔ Print Friendly
Designed specifically for A4 printing.
